What to install before your Operator Intensive

A short, IT-friendly checklist of what needs to be on your laptop for the one-day session. Nothing here is unusual or invasive — these are standard, widely-used developer tools, and everything runs locally on your machine.

~15 minutes to install · Works on Windows & macOS · No servers, no admin daemons
For the executive

On the day, we install a private "AI workforce" — about a hundred specialist agents — that runs on your own laptop inside Anthropic's Claude Code. To make the day productive, your machine needs a handful of free, standard tools in place beforehand. Forward this page to whoever manages your laptop; the next column is written for them.

For IT / Security

The workforce is a set of human-readable instruction files plus small Python helper scripts that run locally inside the Claude Code CLI. There is no server, no inbound connection, and no background service. Required software is listed below, followed by the network and data-handling profile.

2.Network & data handling — the part IT cares about

Connections

  • Outbound HTTPS only, to two destinations: api.anthropic.com (the AI model) and github.com (your repository).
  • No inbound connections. No listening ports. No background service or daemon is installed or run.

Data

  • The agents run locally. The only data that leaves the laptop is (a) the text you choose to send to Anthropic's model, governed by Anthropic's commercial terms / DPA (Anthropic does not train its models on commercial/API customer data), and (b) ordinary Git operations to your private repository.
  • Nothing is sent to ProductBeacon beyond what you explicitly choose to share with us.
  • The workforce files are plain, auditable Markdown and Python — your team can read exactly what they do.

3.Install steps

  1. Install Node.js LTS from nodejs.org.
  2. Install Claude Code: run npm install -g @anthropic-ai/claude-code in a terminal.
  3. Install Git (git-scm.com) and Python 3.10+ (python.org).
  4. Sign in to Claude (your Pro/Max subscription) or set your Anthropic API key.
  5. Accept the GitHub invitation we send and clone your ProductBeacon AI Workforce repository.
  6. Run python install.py once to set up the workforce in your working folder.
That's the whole list. If steps 1–4 are done before the session, we complete steps 5–6 together and spend the day teaching you to operate the workforce. The morning after, you open your laptop and it's waiting for your instructions.
